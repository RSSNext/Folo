よく言われるように、あなたの思考はあなたが読んだものによって決まります。そして、私たちは長い間、雑音だらけのフィードを消費しすぎてきました！Foloはコンテンツを1つのタイムラインに整理し、重要な情報をノイズなくお届けします。リストを共有したり、コレクションを閲覧したり、気を散らすことなくブラウジングを楽しめます。

コミュニティによって管理されている次の方法を使用してインストールすることもできます。

- Arch Linuxを使用している場合は、[timochan](https://github.com/ttimochan)と[grtsinry43](https://github.com/grtsinry43)によってメンテナンスされているパッケージ[folo-appimage](https://aur.archlinux.org/packages/folo-appimage)をインストールできます。
- Nix を使用している場合は、[iosmanthus](https://github.com/iosmanthus) によってメンテナンスされているパッケージ [follow](https://github.com/NixOS/nixpkgs/blob/master/pkgs/by-name/fo/follow/package.nix) をインストールできます。
- macOSで[Homebrew](https://brew.sh)を使用している場合は、[realSunyz](https://github.com/realSunyz)がメンテナンスしているcask [folo](https://formulae.brew.sh/cask/folo)をインストールできます。
- Windowsで[Scoop](https://scoop.sh)を使用している場合は、[cscnk52](https://github.com/cscnk52)が管理するマニフェスト[folo](https://github.com/cscnk52/cetacea/blob/master/bucket/folo.json)をインストールできます。

## ✨ 機能

### カスタマイズされた情報ハブ

幅広いフィードと厳選されたリストを購読できます。お気に入りを厳選して、あなたにとって最も重要なことを常に把握しましょう。

![](https://github.com/user-attachments/assets/11dc7d21-f5d8-4e41-9269-24fc352aa02b)

### 指先で操作できるAI

翻訳、要約などの AI を活用した機能により、よりスマートで効率的なブラウジングが実現します。

![](https://github.com/user-attachments/assets/37cf4f2f-4c5e-4775-86e8-2fa1a1b2ecf5)

### 動的コンテンツのサポート

コンテンツは単なるテキストではないことを私たちは知っています。記事から動画、画像から音声まで、Foloならあらゆるコンテンツを網羅できます。

![](https://github.com/user-attachments/assets/d1379fd6-8767-476e-b0dc-d61753715e26)

### 単なるアプリ以上のもの

これは単なるアプリではありません。Foloはコミュニティであり、オープン性とコミュニティ主導の体験という新しい時代を切り開きます。

![](https://github.com/user-attachments/assets/62004a04-eaea-4f5d-bfbf-4e68b6b90286)
